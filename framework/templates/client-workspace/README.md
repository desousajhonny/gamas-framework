# Client Workspace Template

Template inicial do workspace de uma empresa AI First.

Este workspace pertence ao cliente e deve ser editavel, legivel por humanos e versionavel em GitHub.

Ele contem:

- manifests estruturados em YAML;
- documentos em Markdown;
- departamentos;
- agentes simulados;
- referencias para skills e workflows;
- second brain;
- metricas;
- estado tecnico system-managed em `.gamas/`.

Managed resources devem ser referenciados por `gamas://`, nao copiados como implementacao completa.
