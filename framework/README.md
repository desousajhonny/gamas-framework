# Framework

Camada publica e distribuivel do Gamas-framework.

Contem:

- contratos publicos de skills, workflows e quality gates;
- standards operacionais;
- templates de workspace do cliente.

Nao contem:

- implementacao completa de managed skills;
- prompts proprietarios;
- heuristicas sensiveis;
- logica interna do managed core.

Recursos proprietarios devem ser referenciados por `gamas://`.
