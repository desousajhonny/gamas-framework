# Packages

Esta pasta prepara os pacotes executaveis e bibliotecas do Gamas-framework.

Ela nao e o workspace do cliente. Ela contem a base tecnica futura do framework: CLI, core, runtime, registry client, schemas, validators e Gamas VS Code Agent.

Regras:

- nao colocar aqui documentos editaveis da empresa do cliente;
- nao copiar managed skills, managed workflows ou managed quality gates para templates abertos;
- manter a separacao entre runtime, contratos publicos e logica proprietaria;
- tratar implementacoes profundas como trabalho futuro.
