# Gamas CLI

CLI futura do Gamas-framework.

Responsabilidades:

- criar o workspace inicial da empresa;
- registrar empresa e produto ativo ou planejado;
- gerar manifests e estrutura leve;
- preparar o workspace para o Gamas no VS Code;
- orientar o usuario a rodar `/setup`.

Este scaffold nao implementa o pacote npm. Os comandos abaixo sao placeholders documentais para a implementacao futura.

Comandos planejados:

- `init`
- `login`
- `setup`
- `validate`
- `doctor`
- `run-workflow`
- `sync`

Regra: a CLI instala e organiza. O diagnostico profundo fica para o Gamas Agent.
