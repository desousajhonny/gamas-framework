# `gamas init`

Cria o workspace inicial da empresa.

Deve:

- perguntar o minimo necessario;
- criar manifests iniciais;
- gerar estrutura leve do Company OS;
- registrar produto ativo ou planejado;
- preparar o proximo passo: abrir VS Code e rodar `/setup`.

Nao deve copiar logica proprietaria completa para o workspace do cliente.
