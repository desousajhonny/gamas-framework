# Runtime

Runtime local futuro do Gamas.

Responsabilidades futuras:

- ler manifests;
- manter estado tecnico;
- executar comandos locais;
- aplicar validacoes basicas;
- resolver referencias gerenciadas via registry/runtime apropriado;
- respeitar approval gates.

Estado tecnico deve ficar em `.gamas/`, separado dos documentos editaveis pelo cliente.
