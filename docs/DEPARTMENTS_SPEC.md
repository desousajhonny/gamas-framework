# Departments Spec

## Pergunta que este documento responde

Como departamentos devem ser representados no workspace?

## Departamentos base

O template inclui:

- strategy;
- product;
- design;
- engineering;
- qa;
- marketing;
- sales;
- customer-success;
- finance;
- operations;
- data;
- legal-risk.

Cada departamento contem:

- `README.md`;
- `ROLE.md`;
- `WORKFLOWS.md`;
- `SKILLS.md`;
- `METRICS.md`.

Departamentos sao areas operacionais, nao implementacoes de agentes autonomos.
