# Second Brain Spec

## Pergunta que este documento responde

O que deve ser registrado no second brain?

## Regra central

O second brain guarda memoria operacional reutilizavel.

Registrar:

- decisoes importantes;
- aprendizados de releases;
- feedbacks relevantes;
- mudancas de estrategia;
- pos-mortems;
- sinais de suporte, vendas e produto.

Nao registrar:

- cache tecnico;
- locks;
- estado interno do runtime;
- segredos.

Estado tecnico pertence a `.gamas/`.
