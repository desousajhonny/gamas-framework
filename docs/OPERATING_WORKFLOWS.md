# Operating Workflows

## Pergunta que este documento responde

Qual fluxo operacional o Gamas-framework deve suportar no futuro?

## Fluxo norte

```txt
decisao estrategica -> roadmap -> escopo -> PRD -> planejamento tecnico -> desenvolvimento -> validacao -> testes -> release -> metricas -> aprendizado no second brain
```

## Regras

- toda etapa critica deve ter dono ou agente simulado;
- toda entrega deve ter criterio de aceite;
- release deve passar por validacao;
- aprendizado relevante deve voltar para o second brain;
- workflows managed devem ser referenciados por contrato.
