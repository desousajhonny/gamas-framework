# CLI Spec

## Pergunta que este documento responde

Qual e o papel da CLI do Gamas-framework?

## Responsabilidade

A CLI instala e organiza. O Gamas Agent entende e preenche.

## Comandos planejados

- `init`: cria o workspace inicial;
- `login`: conecta usuario ao registry ou managed core futuro;
- `setup`: orienta o setup conversacional;
- `validate`: valida estrutura, manifests e contratos;
- `doctor`: diagnostica problemas locais;
- `run-workflow`: executa ou conduz workflow;
- `sync`: sincroniza contratos e metadados.

O spec detalhado do wizard esta em `docs/product/CLI_WIZARD.md`.
